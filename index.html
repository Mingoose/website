PennOS

PennOS is a Unix-like operating system written in C that runs on top of Linux. It includes PennShell, a shell with many provided built-ins for redirection, piping, process creation and management, and file system access. PennOS additionally includes a kernel that provides a partial implementation of the POSIX API, using the ucontext library to provide multithreading capabilities to users. In addition, a FAT filesystem is available, with a user-level API for creating files/executables, managing permissions, renaming/moving files, etc.
